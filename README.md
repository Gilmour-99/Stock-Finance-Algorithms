# Stock-Finance-Algorithms
Algorithms of Stock &amp; Finance Models

* 初版功能

  1、计算5/10/20日均线

  2、基于ARIMA进行股票序列预测

  3、基于ARIMA的期望收益率计算

  4、历史平均收益率计算

  5、求解马科维茨前沿组合

  6、根据马科维茨前沿组合给出股票购买计划（向下取整）
